> **現在のバージョン**: v1.6.0
> **キャッチコピー**: 進化心理学で先延ばしを克服。

**プライマル・フォーカス (PrimalFocus)** は、進化心理学に基づいた多言語対応の「先延ばし克服」ツールです。単純なToDoリストではなく、AIがあなたの行動表現型（「アーキタイプ」）を分析し、タスクを低エネルギーの「マイクロステップ」に分解します。そして、過激な心理戦略を用いて、あなたを行動不能状態から実行状態へと移行させます。

---

## ✨ 主な機能 (Key Features)

### 1. 🧬 心理プロファイリング (Psychological Profiling)
*   **30項目の行動分析**: あなたの先延ばしスタイル（例：完璧主義者、反抗者、低エネルギー）を特定します。
*   **AI生成レポート**: 「根源的な恐怖」、強み、弱点、および推奨される心理プロトコルを含む詳細なプロファイルを生成します。
*   **アーキタイプ・システム**: あなたに原始的なアーキタイプ（例：「完璧主義のハンター」）を割り当てます。

### 2. 🧠 AI 戦術計画 (AI Tactical Planning)
*   **曖昧から具体的へ**: 抽象的な目標（例：「論文を書く」）を、脅威のない具体的なマイクロタスクに瞬時に変換します。
*   **戦略の注入**: AIは、開始のハードルを下げるために、プロファイルに基づいた特定の心理的「戦略」（例：「ゴミ下書きプロトコル」）を割り当てます。
*   **ミッションの精査**: 進行中のミッションに対して「情報の補足/調整」を行うことができます。AIに「圧倒されている」「状況が変わった」と伝えると、残りの計画を再調整します。

### 3. ⚔️ 実行モード (Execution Mode)
*   **フォーカス・タイマー**: シングルタスクに没頭するための、気が散らないインターフェース。
*   **パニックボタン (SOS)**: 行き詰まったり不安になったりした場合は、パニックボタンを押してください。AIは現在のタスクを即座に3つの「笑えるほど簡単な」マイクロステップ（例：「ドキュメントを開いて1分間見つめる」）に分解し、行動麻痺を打破します。
*   **ニューラル環境音 (Neural Ambience)**: 以下のサウンドを提供するオーディオエンジンを統合：
    *   ホワイトノイズ (White Noise)
    *   ピンクノイズ/雨音 (Pink Noise)
    *   バイノーラルビート (40Hz ガンマ波 - 集中用)

### 4. 🛡️ ミッションコントロール (Mission Control)
*   **データ主権**: すべてのデータはブラウザのローカル (`localStorage`) に保存され、プライバシーが保護されます。
*   **バックアップシステム**: データ移行やバックアップのための完全なJSONインポート/エクスポート機能。
*   **ゲーミフィケーション**: タスク完了時にドーパミン受容体を飽和させるための視覚的な儀式。
*   **アーカイブシステム**: 過去の勝利を振り返り、アーカイブされた古いミッションを再開することができます。

### 5. 🌐 多言語サポート (Multi-Language Support)
アプリケーションはシステム言語を自動的に検出しますが、手動で切り替えることも可能です。
*   **🇺🇸 English** (英語)
*   **🇨🇳 简体中文** (中国語)
*   **🇯🇵 日本語**
*   **🇰🇷 한국어** (韓国語)

## 🛠 技術スタック (Tech Stack)

*   **フロントエンド**: React 18, TypeScript, Vite
*   **スタイリング**: Tailwind CSS (サイバーパンク/ダークモード テーマ)
*   **AI**: Google Gemini API (`gemini-3-flash-preview`)
*   **アイコン**: Lucide React
*   **状態管理/ストレージ**: React Hooks + LocalStorage

## 📅 更新履歴 (Changelog)

### v1.6.0 (Current)
*   **新規**: **オンボーディングシステム**。評価、ダッシュボード、計画、実行ビューを通じてユーザーをガイドする「スポットライト」式のチュートリアルを追加しました。
*   **新規**: **言語設定ウィザード**。初回ユーザー向けに、専用の全画面言語選択インターフェースを追加しました。
*   **修正**: **評価フォームの安定性**。素早いクリックやダブルクリックによる「質問スキップ」やインデックス範囲外エラーを防ぐため、入力ロック機構を追加しました。
*   **修正**: **チュートリアルの位置調整**。モバイルや小さなウィンドウでツールチップが画面外に飛び出さないよう、位置決めロジックを改善しました。
*   **修正**: 条件付きフックの実行によって引き起こされるReactハイドレーションエラー (`Minified React error #300`) を解決しました。

### v1.5.0
*   **機能**: **パニックボタン**。実行モードに緊急ボタンを追加しました。ユーザーが行き詰まった場合、AIが介入してタスクを分解します。
*   **機能**: **オーディオエンジン**。Web Audio APIシンセサイザーを統合し、ホワイトノイズ、ピンクノイズ、バイノーラルビートをサポートしました。
*   **機能**: **ミッションの精査**。アクティブなミッションを「編集/調整」する機能を追加しました。ユーザーが新しいコンテキストを提供すると、AIは完了したタスクを保持したまま、残りの計画を再生成します。

### v1.4.0
*   **機能**: **ローカライズ**。簡体字中国語、日本語、韓国語を完全にサポートしました。
*   **UI**: 初回評価後に詳細な心理分析レポートをいつでも表示できる `ProfileModal` を追加しました。

### v1.3.0
*   **機能**: **データの移植性**。システム全体のバックアップ用 `インポート/エクスポート` ボタン（JSON形式）を追加しました。
*   **機能**: **単一ミッションのインポート**。個々のミッションファイルの共有/インポート機能を追加しました。
*   **UX**: 「グローバル保存」ショートカット (`Ctrl+S`) を追加しました。

### v1.2.0
*   **機能**: **実行ロジック**。フォーカスタイマー、タスク完了ロジック、および「ミッション完了」時の祝賀画面を実装しました。
*   **UX**: タスクリストにAIが推奨する「心理学的オーバーライド戦略」を表示するようにしました。

### v1.1.0
*   **バックエンド**: Google Gemini API を統合しました。
*   **機能**: 30問の心理評価ロジックと、心理プロファイリングのためのプロンプトエンジニアリングを実装しました。

### v1.0.0
*   初回リリース。
*   基本的なプロジェクトCRUD（作成、読み取り、更新、削除）。
*   サイバーパンクUIの実装。
