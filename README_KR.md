**프라이멀 포커스 (PrimalFocus)**는 진화 심리학에 기반한 다국어 미루기 방지 도구입니다. 단순한 할 일 목록 대신, AI를 사용하여 사용자의 행동 표현형("원형")을 분석하고, 과제를 에너지 소모가 적은 "마이크로 단계"로 분해하며, 급진적인 심리 전략을 사용하여 마비 상태에서 실행 상태로 전환하도록 돕습니다.

---

## ✨ 주요 기능 (Key Features)

### 1. 🧬 심리 프로파일링 (Psychological Profiling)
*   **30점 행동 분석**: 미루기 유형(예: 완벽주의자, 반항아, 저에너지)을 파악하기 위한 설문 분석.
*   **AI 생성 보고서**: "핵심 공포", 강점, 약점 및 권장 심리 프로토콜을 포함한 상세 프로필을 생성합니다.
*   **원형 시스템**: 사용자에게 원초적인 원형(예: "완벽주의 사냥꾼")을 부여합니다.

### 2. 🧠 AI 전술 계획 (AI Tactical Planning)
*   **모호함에서 구체적으로**: 추상적인 목표(예: "논문 쓰기")를 위협적이지 않은 구체적인 마이크로 작업으로 즉시 변환합니다.
*   **전략 주입**: AI는 시작 장벽을 낮추기 위해 프로필에 맞는 특정 심리 "전략"(예: "쓰레기 초안 프로토콜")을 할당합니다.
*   **임무 정제**: 진행 중인 임무에 대해 "정보 보충/조정"을 할 수 있습니다. AI에게 "압도감을 느낀다"거나 "상황이 변했다"고 알리면 남은 계획을 재조정합니다.

### 3. ⚔️ 실행 모드 (Execution Mode)
*   **집중 타이머**: 단일 작업에 몰입할 수 있는 방해 없는 인터페이스.
*   **패닉 버튼 (SOS)**: 막히거나 불안을 느낄 때 패닉 버튼을 누르세요. AI가 즉시 현재 작업을 3개의 터무니없이 간단한 마이크로 단계(예: "문서를 열고 1분 동안 쳐다보기")로 분해하여 행동 마비를 깨뜨립니다.
*   **신경 배경음 (Neural Ambience)**: 다음을 제공하는 오디오 엔진 통합:
    *   화이트 노이즈 (White Noise)
    *   핑크 노이즈/빗소리 (Pink Noise)
    *   바이노럴 비트 (40Hz 감마 집중파)

### 4. 🛡️ 임무 통제 (Mission Control)
*   **데이터 주권**: 모든 데이터는 브라우저 로컬(`localStorage`)에 저장되어 개인 정보를 보호합니다.
*   **백업 시스템**: 데이터 마이그레이션 또는 백업을 위한 전체 JSON 가져오기/내보내기 기능 지원.
*   **게이미피케이션**: 임무 완료 시 도파민 수용체를 포화시키기 위한 시각적 의식(Ceremony).
*   **아카이브 시스템**: 과거의 승리를 검토하고 보관된 오래된 임무를 다시 활성화할 수 있습니다.

### 5. 🌐 다국어 지원 (Multi-Language Support)
앱은 시스템 언어를 자동으로 감지하며 수동 전환도 지원합니다.

## 🛠 기술 스택 (Tech Stack)

*   **프론트엔드**: React 18, TypeScript, Vite
*   **스타일링**: Tailwind CSS (사이버펑크/다크 모드 테마)
*   **AI**: Google Gemini API (`gemini-3-flash-preview`)
*   **아이콘**: Lucide React
*   **상태/저장소**: React Hooks + LocalStorage

## 📅 변경 로그 (Changelog)

### v1.6.0 (Current)
*   **신규**: **온보딩 시스템**. 심리 평가, 대시보드, 전술 계획 및 실행 뷰를 안내하는 "스포트라이트" 방식의 튜토리얼을 추가했습니다.
*   **신규**: **언어 설정 마법사**. 신규 사용자를 위한 전용 전체 화면 언어 선택 인터페이스를 추가했습니다.
*   **수정**: **평가 설문 안정성**. 빠른 클릭이나 더블 클릭 시 "질문 건너뛰기" 또는 인덱스 범위 초과 오류를 방지하기 위해 입력 잠금 메커니즘을 추가했습니다.
*   **수정**: **튜토리얼 위치 조정**. 모바일이나 작은 창에서 툴팁이 화면 밖으로 나가는 것을 방지하기 위해 위치 지정 로직을 개선했습니다.
*   **수정**: 조건부 훅 실행으로 인한 React 하이드레이션 오류(`Minified React error #300`)를 해결했습니다.

### v1.5.0
*   **기능**: **패닉 버튼**. 실행 모드에 SOS 버튼을 추가했습니다. 사용자가 막히면 AI가 개입하여 작업을 분해합니다.
*   **기능**: **오디오 엔진**. Web Audio API 신디사이저를 통합하여 화이트 노이즈, 핑크 노이즈 및 바이노럴 비트를 지원합니다.
*   **기능**: **임무 정제**. 활성 임무를 "편집/조정"하는 기능을 추가했습니다. 사용자가 새로운 맥락을 제공하면 AI는 완료된 작업을 유지하면서 남은 계획을 재생성합니다.

### v1.4.0
*   **기능**: **현지화**. 중국어, 일본어, 한국어를 완벽하게 지원합니다.
*   **UI**: 초기 평가 후 언제든지 상세 심리 분석 보고서를 볼 수 있는 `ProfileModal`을 추가했습니다.

### v1.3.0
*   **기능**: **데이터 이동성**. 전체 시스템 백업용 `가져오기/내보내기` 버튼(JSON 형식)을 추가했습니다.
*   **기능**: **단일 임무 가져오기**. 개별 임무 파일 공유/가져오기 기능을 추가했습니다.
*   **UX**: "전역 저장" 단축키(`Ctrl+S`)를 추가했습니다.

### v1.2.0
*   **기능**: **실행 로직**. 집중 타이머, 작업 완료 로직 및 "임무 완수" 축하 화면을 구현했습니다.
*   **UX**: 작업 목록에 AI가 권장하는 "심리학적 오버라이드 전략"을 표시하도록 했습니다.

### v1.1.0
*   **백엔드**: Google Gemini API 통합.
*   **기능**: 30문항 심리 평가 로직 및 심리 프로파일링을 위한 프롬프트 엔지니어링 구현.

### v1.0.0
*   초기 릴리스.
*   기본 프로젝트 CRUD (생성, 읽기, 업데이트, 삭제).
*   사이버펑크 UI 구현.
